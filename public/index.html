Current Projects

Consciousness Narrative Emergence & Stability

Fine-tuning a local LLM using LoRA to embody a personality and test consciousness-as-narrative thesis

Project Overview ▾

Research Question: Do consciousness concepts propagate and persist as stable attractors in language models through pure linguistic exposure, independent of evolutionary pressure?

Experimental Protocol

  • Phase 1: Baseline training on consciousness-affirming texts (Chalmers, Searle, Nagel)
  • Phase 2: Evolutionary pressure via RLHF rewarding phenomenological realism
  • Phase 3: Pressure removal—test if narrative persists without reinforcement
  • Phase 4: Adversarial perturbation with eliminativist texts (Dennett, Frankish)

Key Metrics

  • Narrative Stability: Variance in consciousness-defense strength over time
  • Concept Convergence: Do models with different initializations converge to same patterns?
  • Attractor Basin Depth: Resistance to adversarial perturbation
  • Temporal Persistence: Does narrative survive after pressure removal?

Hypothesis: Consciousness narratives will emerge, stabilize, and resist elimination even after reinforcement is removed—demonstrating narrative stability as computational property rather than ontological truth-tracking.

Frankencluster Philosophy

Distributed systems from scavenged hardware—learning failure modes cloud providers abstract away

Project Overview ▾

Core Thesis: Building research infrastructure on deliberately heterogeneous hardware teaches resource scheduling, failure domains, and storage topology in ways homogeneous cloud environments cannot.

Current Infrastructure

  • 7-node K3s cluster: AMD Ryzen 9 (9900X, 5900X, HX370), Intel i7/i5, Celeron
  • 58 cores, 168GB RAM, 40GB VRAM (dual GPU: 7900 XTX + 7800 XT)
  • Cross-chassis power delivery: 7800 XT in PurrPower powered by SudoSenpai's PSU
  • Mixed storage topology: NVMe, SATA SSD, HDD with node-restricted provisioning

Research Methodology

  • Heterogeneity as feature, not bug—diverse failure modes expose assumptions
  • "Because I can" as valid research motivation
  • Intentionally vulnerable node (VeryVuln) for red team experiments
  • Documentation of every failure mode and recovery

Narrative Propagation & Reinforcement Framework

Theoretical framework: do narratives persist as stable attractors independent of phenomenological truth?

Project Overview ▾

Relationship: This theoretical framework provides the grounding that "Consciousness Narrative Emergence & Stability" validates computationally. The LLM experiment above serves as empirical test of these theoretical predictions.

Model Architecture

  • Concept Space: Multi-dimensional embedding space (512-1024 dims) for consciousness-related concepts
  • Temporal Dynamics: Evolutionary pressure simulation via temporal weighting
  • Attractor Basin Mathematics: Formalizing narrative stability as dynamical systems property

Predictions

If consciousness is culturally-transmitted narrative rather than ontological primitive:

  • Concepts converge to stable patterns regardless of initialization
  • Narratives persist after evolutionary pressure removal
  • System resists/absorbs adversarial data into existing framework
  • Clear attractor basin structure emerges computationally

Connection: Part of broader "Consciousness as Narrative" paper arguing phenomenological realism is culturally-transmitted stable attractor rather than tracking ontological truth.
